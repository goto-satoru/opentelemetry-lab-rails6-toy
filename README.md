# Rails 'Toy' appplication tracing with OpenTelemetry

* Ruby 2.7.6

## Prerequisites

- Rails 6
- Docker 
- Yarn

## Deployment

- docker-compose up
- bundle install
- rails db:migrate
- rails webpacker:install
- rails server
